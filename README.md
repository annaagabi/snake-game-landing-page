# 🎮 Sobre o Projeto

O Snake Game é um dos jogos mais clássicos da história dos videogames. Esta Landing Page foi criada com o objetivo de apresentar o jogo de forma moderna e atrativa, utilizando conceitos atuais de UI/UX Design, responsividade e interatividade.

A proposta visual combina elementos retrô dos jogos arcade com uma identidade futurista inspirada em Cyberpunk e Synthwave, proporcionando uma experiência visual imersiva.

---

# 🎯 Objetivo

Desenvolver uma Landing Page responsiva para divulgação do Snake Game, aplicando boas práticas de desenvolvimento front-end e design de interfaces.

### Objetivos específicos:

* Criar uma interface moderna e intuitiva;
* Aplicar HTML semântico;
* Utilizar CSS moderno com Flexbox e Grid Layout;
* Implementar interações com JavaScript Vanilla;
* Garantir responsividade para dispositivos móveis, tablets e desktops;
* Trabalhar conceitos de UI/UX focados em experiência do usuário.

# 🚀 Como Executar Localmente

1. Clone o repositório:

```bash
git clone https://github.com/annaagabi/snake-game-landing-page.git
```

2. Acesse a pasta do projeto:

```bash
cd snake-game-landing-page
```

3. Abra o arquivo:

```bash
index.html
```

Ou utilize a extensão Live Server no Visual Studio Code.
